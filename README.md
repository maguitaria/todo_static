# To Do List
This project is simple to-do list app for adding, editing, deleting tasks.
Using Typescript basics, Vite tool and HTML, CSS.
# Installation
1. Clone the repo https://github.com/maguitaria/typescript_toDo_list.git
2. Run **npm run dev**
3. Enjoy! :wink:
